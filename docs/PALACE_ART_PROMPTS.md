# Memory Palace — painted-image prompt pack

The palace currently renders hand-drawn vector scenes (engraved gold line-art). To upgrade
any/all rooms to painted raster images: generate each image below (Canva Magic Media,
Midjourney, DALL-E, etc.), save as JPG at ~1200x700 (16:9-ish), drop into **/palace-art/**
with the EXACT filename, then set `USE_ROOM_IMAGES = true` in `palace.html`. Rooms fall back
per-file is NOT automatic — the flag switches all rooms, so add all 30 before flipping it.

**Style suffix — append to every prompt:** "painterly digital illustration, deep navy blue
and warm gold palette, dramatic chiaroscuro candlelight, atmospheric, moody, no text, no
words, no lettering anywhere"
